# 电视测试卡生成器 (TV Test Card Generator)

一个简单的在线工具，用于生成各种电视测试卡图案。

## 功能

*   多种内置测试图案 (彩条, SMPTE, PM5544 等)。
*   可添加几何图形 (圆形, 过扫描区域)。
*   可自定义标签文本，并显示当前时间。
*   可调整文本样式 (反转颜色, 加框, 闪烁)。
*   支持 4:3 和 16:9 预览比例。
*   生成可用于 OBS 等直播软件的浏览器源链接。

## 使用说明

1.  打开 `index.html` 文件。
2.  在左侧控制面板中选择您喜欢的图案和选项。
3.  **点击预览显示器**或点击“全屏显示”按钮，以全屏显示测试卡。再次点击可返回。
4.  **复制全屏链接**，并在 OBS 中添加一个浏览器源，将此链接粘贴进去，即可在您的视频中使用自定义的测试卡。
